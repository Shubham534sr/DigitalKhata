# 🚀 DigitalKhata

DigitalKhata is a full-stack business management and digital ledger application designed to help small businesses manage customers, transactions, payments, collections, and reports efficiently.

## ✨ Features

- 👥 Customer Management
- 💳 Credit & Debit Tracking
- 💰 Payment Management
- 📊 Dashboard & Reports
- 📦 Shopping Module
- 📑 Transaction History
- 🔐 User Authentication
- 📱 Responsive User Interface

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- CSS3
- JavaScript (ES6)

### Backend
- Node.js
- Express.js

### Database
- MySQL

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Shubham534sr/DigitalKhata.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
